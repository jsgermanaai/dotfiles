# 🚀 Tmux Cheat Sheet - Modern Configuration

## 🔑 Prefix Key
**Prefix**: `Ctrl-a` (instead of default `Ctrl-b`)

## 🚀 Getting Started

### Basic Commands
```bash
tmux                    # Start new session
tmux new -s myname      # Start new session with name
tmux a                  # Attach to last session
tmux a -t myname        # Attach to named session
tmux ls                 # List sessions
tmux kill-session -t myname  # Kill named session
```

## ⌨️ Key Bindings

### 📋 Session Management
| Key | Action |
|-----|--------|
| `Ctrl-a S` | Create new session |
| `Ctrl-a $` | Rename current session |
| `Ctrl-a s` | Show session list |
| `Ctrl-a (` | Previous session |
| `Ctrl-a )` | Next session |
| `Ctrl-a d` | Detach session |

### 🪟 Window Management
| Key | Action |
|-----|--------|
| `Ctrl-a c` | Create new window |
| `Ctrl-a ,` | Rename current window |
| `Ctrl-a &` | Kill current window |
| `Ctrl-a X` | Kill current window (custom) |
| `Shift-Left` | Previous window (no prefix!) |
| `Shift-Right` | Next window (no prefix!) |
| `Ctrl-a 0-9` | Switch to window number |

### 🔲 Pane Management
| Key | Action |
|-----|--------|
| `Ctrl-a \|` | Split horizontally |
| `Ctrl-a -` | Split vertically |
| `Ctrl-a x` | Kill current pane |
| `Alt-Left/Right/Up/Down` | Navigate panes (no prefix!) |
| `Ctrl-a h/j/k/l` | Navigate panes (Vim-style) |
| `Ctrl-a H/J/K/L` | Resize panes (Vim-style) |
| `Ctrl-a e` | Toggle synchronize panes |

### 📝 Copy Mode & Clipboard
| Key | Action |
|-----|--------|
| `Ctrl-a [` | Enter copy mode |
| `Ctrl-a ]` | Paste buffer |
| `v` | Start selection (in copy mode) |
| `y` | Copy selection (in copy mode) |
| `r` | Rectangle selection (in copy mode) |

### 🔧 Configuration & Utility
| Key | Action |
|-----|--------|
| `Ctrl-a r` | Reload tmux config |
| `Ctrl-a Ctrl-e` | Edit tmux config |
| `Ctrl-k` | Clear screen and history (no prefix!) |
| `Ctrl-a ?` | List all key bindings |

## 🔌 Plugin Management (TPM)

### Installing Plugins
1. Add plugin to `~/.tmux.conf`
2. Press `Ctrl-a I` (capital i) to install
3. Press `Ctrl-a U` to update plugins
4. Press `Ctrl-a alt-u` to uninstall plugins

### Included Awesome Plugins
- **tmux-resurrect**: Save/restore tmux sessions
- **tmux-continuum**: Auto-save sessions every 15 minutes
- **vim-tmux-navigator**: Seamless navigation between vim and tmux
- **tmux-yank**: Copy to system clipboard
- **tmux-copycat**: Enhanced search in copy mode
- **tmux-open**: Open highlighted files/URLs

## 🚀 Pro Tips

### 🎯 Quick Workflows
```bash
# Start a development session
tmux new -s dev
Ctrl-a |        # Split vertically
Ctrl-a -        # Split horizontally in right pane
# Now you have: editor | terminal
#                      | logs

# Navigate between panes with Alt+arrows (no prefix needed!)
```

### 🔍 Search in Copy Mode
1. `Ctrl-a [` to enter copy mode
2. `Ctrl-s` to search forward
3. `Ctrl-r` to search backward
4. `n` for next match, `N` for previous

### 📋 System Clipboard Integration
- `y` in copy mode automatically copies to macOS clipboard
- Works with LazyVim's clipboard settings
- Paste anywhere with `Cmd+V`

### 🔄 Session Resurrection
```bash
# Save session manually
Ctrl-a Ctrl-s

# Restore session manually  
Ctrl-a Ctrl-r
```

## 🔧 Customization

Your config is at `~/.tmux.conf`. Some popular customizations:

```bash
# Change prefix key back to Ctrl-b
set -g prefix C-b

# Different split keys
bind '\' split-window -h
bind '_' split-window -v

# Custom status bar
set -g status-right "🚀 #H | %Y-%m-%d %H:%M"
```

## 🐛 Troubleshooting

### Colors not working?
```bash
# Check your TERM
echo $TERM

# Should be: tmux-256color or screen-256color
# If not, restart tmux or your terminal
```

### Plugins not loading?
```bash
# Install TPM if missing
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm

# Then in tmux: Ctrl-a I
```

### Key bindings not working?
```bash
# Reload config
Ctrl-a r

# Or restart tmux
tmux kill-server
tmux
```
